# RentCar Real Commercial

Bu paket demo HTML değil; Node.js + Express + SQLite + Admin Panel + Frontend içeren çalıştırılabilir araç kiralama sistemidir.

## Kurulum

```bash
npm install
cp .env.example .env
npm run seed
npm start
```

## Adresler

- Site: http://localhost:3000
- Admin: http://localhost:3000/admin

## Admin Bilgileri

- E-posta: admin@rentcar.com
- Şifre: admin123

## İçerik

- Kullanıcı kayıt/giriş
- JWT oturum sistemi
- Admin yetki kontrolü
- Araç ekleme
- Araç düzenleme
- Araç silme
- Fiyat/fotoğraf/model/kategori yönetimi
- Rezervasyon oluşturma
- Rezervasyon onay/iptal
- Kullanıcı listeleme
- Dashboard istatistikleri
- WhatsApp bağlantısı
- SQLite veritabanı
- Mobil uyumlu tasarım

## Canlıya alma

Node.js destekli VPS veya hosting gerekir. cPanel statik hosting yetmez çünkü backend çalışır.
